# Sharvari Nilesh Dhekre Portfolio

Premium responsive portfolio website built with HTML, CSS, and JavaScript.

## Deploy
Upload `index.html`, `style.css`, `script.js`, `README.md`, and the `assets` folder to your GitHub repository root.

## Add photo
Add your professional photo inside `assets` and name it exactly:

```text
profile.jpg
```

## GitHub Pages
Settings → Pages → Deploy from branch → main → /root.

Live URL:

```text
https://sharvarid03.github.io/Sharvari-Portfolio/
```
